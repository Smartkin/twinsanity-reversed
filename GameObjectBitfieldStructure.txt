Object bitfield structure:
(bitfield >> 0x14 & 0xFF) - InstanceNode type // valid InstanceNode type is between 0 and 8
// Same InstanceNode types are
// 0,3,4,5
// 2,6,7 (only difference is that these are missing a structure in a field)
(bitfield >> 0xC & 0xFF) - Unknown // when InstanceNode type is 1 must be 0x10 or 0x11 else considered as InstanceNode type 2,6,7
(bitfield >> 0x6 & 0x3F) - UnkArray size when it comes to OGI stuff
(bitfield & 0x3F) - UnkArray size when it comes to OGI stuff
